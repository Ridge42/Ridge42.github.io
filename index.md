## 📈 My Projects

### **Predicting Movie Preferences with Neural Networks and Learned Representations**
This project explored whether we can accurately predict if a user will like a movie by combining engineered recommendation features with learned representations from a Two Towers model. I implemented and compared logistic regression, KNN, and a neural network with user/movie embeddings, while designing a leakage-free validation pipeline and tuning models using F1 score evaluation.

The results showed that the neural network substantially outperformed the baseline methods (F1 = 0.897), demonstrating that learned embeddings and affinity representations capture meaningful nonlinear relationships in user preferences and significantly improve recommendation performance.

Check the github for the code and you can look at [this page](https://github.com/chamiltonb8/STAT_486_final_proj/blob/main/progress/04_supervised.md) for a more detailed explanation of the project.

### **Analysis of Reported Rat Problems in New York - Placed 2nd in Case Competition**
For this case competition, we were tasked to discover what districts in NY had the most rat complaints, most "actual" cases of rats, and factors that contributed to rats appearing in the districts. We were also tasked with coming up with ideas for solutions for the rat problems given our findings. 

We made heat maps with areal data, made a linear regression and random forest model, and visualization tools to showcase our findings.
